Detailed Description

Core C++ library for OpenX Manipulator kinematics.

This library implements forward and inverse kinematics calculations for the OpenX Manipulator. It provides functionality for:

  • Forward kinematics using DH parameters
  • Inverse kinematics using both geometric and Newton-Raphson methods
  • End-effector pose calculations and joint angle computations
