Detailed Description

ROS 2 wrapper for manipulator forward kinematics calculations.

This node provides a ROS 2 interface for forward kinematics calculations. It subscribes to joint angles, uses the manipulator_core library to compute forward kinematics, and publishes the end-effector pose.
